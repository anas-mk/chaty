# Chat App

A real-time chat application built with Flutter, powered by Firebase Authentication and Cloud Firestore, following a clean and scalable architecture (Repository / Service / Entity / Model).
The app allows users to send and receive messages instantly, manage authentication, and enjoy a smooth modern UI based on Bloc state management.

# Features

- Real-time Messaging using Firestore streams

- User Authentication via FirebaseAuth

- Clean Architecture (Repository → Service → Model → Entity)

- Bloc/Cubit for state management

- Send text messages with unique IDs (UUID)

- Responsive UI and smooth experience

- Fast, scalable, and easy to maintain codebase



# Tech Stack

- Flutter

- Dart

- Firebase Authentication

- Cloud Firestore

- Bloc / Cubit

- UUID package
